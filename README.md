# 🧼 Laundry & Household Care App(NestWash)

A mobile application built with **Flutter** and **Ruby on Rails** to streamline the booking and management of **laundry** and **home cleaning services**. The app is designed to provide a smooth and intuitive experience for users while offering an efficient backend for service providers.

---

## 📱 Features

- 🔐 User authentication & onboarding
- 🧺 Book laundry or household cleaning services
- 📅 Schedule and manage service bookings
- 🔔 Real-time updates and notifications
- 💳 Secure payment integration (payment gateway)
- 📜 View service history and track past bookings
- 👥 Role-based access for customers and service providers

---

## ⚙️ Tech Stack

| Layer         | Technology           |
|---------------|--------------------- |
| Frontend      | Flutter              |
| Backend       | Ruby, Ruby on Rails  |
| APIs          | REST                 |
| State Management | Riverpod          |
| Authentication | Google Auth         |
| Storage       | Postgres             |
